# codsoftThis repository contains three beginner-friendly Python projects that demonstrate core programming concepts such as user input, conditionals, loops, and randomization.

 Rock Paper Scissors Game

A command-line game where the user plays against the computer. The program randomly selects a choice for the computer and compares it with the user's input to determine the winner. It also keeps track of scores and allows multiple rounds.

 Simple Calculator

A basic calculator that performs arithmetic operations like addition, subtraction, multiplication, and division. The user inputs two numbers and an operator, and the program displays the result accordingly.

 Password Generator

A random password generator that creates secure passwords based on a user-defined length. Each password includes a mix of letters, numbers, and special characters, ensuring better security.
